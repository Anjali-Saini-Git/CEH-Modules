# Module 3 – Network Scanning

## Topics Covered

### Network Scanning Fundamentals
- What is Network Scanning?
- TCP Communication Flags (SYN, ACK, FIN, RST, URG, PSH)
- Three-Way Handshake

### Host Discovery Techniques
- **Netdiscover** – ARP reconnaissance tool
- **Advanced IP Scanner** – GUI-based scanning
- **ICMP & ARP** – Ping sweeps for live host detection

### Nmap (Network Mapper)
- Service version detection (`-sV`)
- Script scanning (`-sC`)
- OS fingerprinting (`-O`)
- Port-specific scanning (FTP, SSH, etc.)

### Firewall / IDS Evasion
- MAC Address Spoofing (`--spoof-mac`)
- Decoy Scans (`-D RND`)

### FTP Lab Report
- Installing vsftpd on Linux
- Configuring anonymous login
- Accessing FTP via Windows CMD
